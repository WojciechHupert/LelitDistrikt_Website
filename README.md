# Lelit Distrikt

Lelit Distrikt is an immersive social experiment and narrative exploration project preserved beneath a dome. 

## Project Overview
This project investigates the intersection of AI, sociology, and digital ethics through a high-fidelity simulation. Residents live within an enclosed Mediterranean town, unaware that the outside world has collapsed. The player takes on the role of an observer, interacting with a persistent world and its inhabitants.

## Core Features
- **Persistent Simulation**: A world that runs in real-time, even when the player is away.
- **AI-Driven Narrative**: Integrated with advanced LLMs (ChatGPT API) for dynamic, emergent character interactions.
- **Authored Characters**: Meet Auggie, the Philosopher, the Painter, and others with deep, interconnected histories.
- **Technical Excellence**: Built with Unreal Engine 5 for high-fidelity visuals and immersive atmosphere.
