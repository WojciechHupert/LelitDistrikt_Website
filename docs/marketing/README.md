# Marketing Documents

This folder contains the marketing-facing source documents for `LelitDistrikt2`.

These files are written to help three audiences:

- collaborators who need a clear understanding of the product
- website and pitch materials that need current-language source copy
- future AI or human editors who must avoid overstating what is already implemented

## Canonical Rules

1. Describe the current product from code and current technical docs, not from older pitch language.
2. Separate `current state`, `active direction`, and `future ambition`.
3. Do not present experimental or partially validated systems as if they are consumer-ready.
4. Use the product name `Lelit Distrikt 2` unless a file is intentionally describing the broader franchise or concept.
